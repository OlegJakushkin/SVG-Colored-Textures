SVG-Colored-Textures

SVG-Colored-Textures is a collection of textures and color filters. We have many color options in D3 (choropleth example). Colors have a standart. Yet we have no such standart for textures with no plane color. This collection of SVG defs and simple js helpers was made to simplify this problem. 153 color**s, **18 textures == 2754 different combinations!

Demo 1 (pure svg), Demo 2 (D3)

Solved D3/SVG problems

We fill objects with color. So fill style field is used. To apply a texture we would need to fill object with so called svg pattern. Therefore, we will have no place for colorization.

Presented solution

fill with predefined pattern, change color via filter!
